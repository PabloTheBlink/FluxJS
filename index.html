JS 
// Crear partículas con configuración por defecto

1 Fondo de página completa ✨

¿Qué hace? Crea un fondo de partículas que cubre toda tu página web, perfecto para landing pages o sitios personales.

📝 HTML - Añade esto a tu página:

HTML 
<!-- Div de fondo para las partículas -->
<div id="fondo-particulas"></div>

<!-- Tu contenido normal aquí -->
<div class="contenido">
  <h1>¡Mi página genial!</h1>
  <p>Contenido con partículas de fondo</p>
</div>

<!-- Script de FluxJS -->
<script src="https://unpkg.com/@pablotheblink/fluxjs@1.0.2/flux.js"></script>

🎨 CSS - Estilos para el fondo:

CSS 
#fondo-particulas {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  z-index: -1; /* Detrás del contenido */
  background: linear-gradient(135deg, #0c0c0c 0%, #1a1a1a 100%);
}

.contenido {
  position: relative;
  z-index: 1; /* Encima de las partículas */
  padding: 2rem;
}

⚡ JavaScript - La magia:

JS 
// Esperar a que la página cargue
document.addEventListener('DOMContentLoaded', function() {
  // Crear las partículas
  const flux = new FluxJS({
    container: document.getElementById('fondo-particulas'),
    count: 80,
    color: '#4fc3f7',
    size: { min: 1, max: 3 },
    connections: { enabled: true, distance: 120 }
  });
});

2 Solo en una sección 🎯

¿Qué hace? Añade partículas solo a una parte específica de tu web, como un hero section o área destacada.

📝 HTML - Estructura básica:

HTML 
<!-- Sección con partículas -->
<section class="hero-section">
  <div id="particulas-hero"></div>
  <div class="hero-content">
    <h1>¡Bienvenido!</h1>
    <p>Esta sección tiene partículas geniales</p>
  </div>
</section>

<!-- Resto de tu página sin partículas -->
<section>
  <h2>Contenido normal</h2>
  <p>Sin partículas aquí</p>
</section>

🎨 CSS - Estilos para la sección:

CSS 
.hero-section {
  position: relative;
  height: 100vh; /* Altura completa de la pantalla */
  overflow: hidden;
}

#particulas-hero {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  z-index: 1;
}

.hero-content {
  position: relative;
  z-index: 2; /* Encima de las partículas */
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  height: 100%;
  text-align: center;
  color: white;
}

⚡ JavaScript - Solo para esa sección:

JS 
document.addEventListener('DOMContentLoaded', function() {
  const flux = new FluxJS({
    container: document.getElementById('particulas-hero'),
    count: 50,
    color: '#ffffff',
    size: { min: 0.5, max: 2 },
    speed: { min: 0.1, max: 0.3 }
  });
});

3 Varios efectos en una página 🌟

¿Qué hace? Crea diferentes efectos de partículas en distintas partes de tu página. ¡Cada sección con su personalidad!

📝 HTML - Múltiples contenedores:

HTML 
<!-- Header con estrellas -->
<header>
  <div id="estrellas-header"></div>
  <h1>Mi Sitio Espacial</h1>
</header>

<!-- Sección de servicios con red -->
<section class="servicios">
  <div id="red-servicios"></div>
  <h2>Nuestros Servicios</h2>
</section>

<!-- Footer con burbujas -->
<footer>
  <div id="burbujas-footer"></div>
  <p>© 2025 Mi Empresa</p>
</footer>

⚡ JavaScript - Tres efectos diferentes:

JS 
document.addEventListener('DOMContentLoaded', function() {
  // Estrellas en el header
  new FluxJS({
    container: document.getElementById('estrellas-header'),
    count: 100,
    color: '#ffffff',
    size: { min: 0.5, max: 1.5 }
  });

  // Red de conexiones en servicios
  new FluxJS({
    container: document.getElementById('red-servicios'),
    count: 30,
    color: '#00ff88',
    connections: { enabled: true, distance: 80 }
  });

  // Burbujas en el footer
  new FluxJS({
    container: document.getElementById('burbujas-footer'),
    count: 20,
    color: '#4FC3F7',
    animation: 'bounce'
  });
});

💡 Consejos de experto

📱

Responsive

Usa menos partículas en móviles para mejor rendimiento: count: window.innerWidth < 768 ? 30 : 80

⚡

Rendimiento

Si tu página va lenta, reduce el número de partículas o desactiva las conexiones.

🎨

Colores

Usa colores que combinen con tu diseño. Los colores claros se ven mejor en fondos oscuros.

📏

Z-index

Usa z-index: -1 para fondos y z-index: 1 para efectos encima del contenido.

Los secretos del control

JS 
{
  container: element, // Contenedor
  count: 50, // Número de partículas
  color: '#ffffff', // Color
  size: { min: 1, max: 3 }, // Tamaño
  speed: { min: 0.1, max: 0.5 }, // Velocidad
  opacity: { min: 0.3, max: 0.8 }, // Opacidad
  connections: { // Conexiones
    enabled: true,
    distance: 100
  },
  mouse: { // Interacción
    interact: true,
    distance: 150
  }
}

Órdenes para dominar

JS 
// Control básico
flux.start(); // Iniciar
flux.stop(); // Detener
flux.destroy(); // Eliminar

// Configuración dinámica
flux.setColor('#ff0000');
flux.setCount(100);
flux.updateConfig({
  speed: { min: 1, max: 2 }
});
