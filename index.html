1 Fondo de página completa ✨

¿Qué hace? Crea un fondo de partículas que abraza toda tu página web, perfecto para landing pages que quieras que sean inolvidables.

📝 HTML - Añade esto a tu página:

HTML 
<!-- Div de fondo para las partículas -->
<div id="fondo-particulas"></div>

<!-- Tu contenido normal aquí -->
<div class="contenido">
  <h1>¡Mi página genial!</h1>
  <p>Contenido con partículas de fondo</p>
</div>

<!-- Script de FluxJS -->
<script src="https://unpkg.com/@pablotheblink/fluxjs@1.0.2/flux.js"></script>

🎨 CSS - Estilos para el fondo:

CSS 
#fondo-particulas {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  z-index: -1; /* Detrás del contenido */
  background: linear-gradient(135deg, #0c0c0c 0%, #1a1a1a 100%);
}

.contenido {
  position: relative;
  z-index: 1; /* Encima de las partículas */
  padding: 2rem;
}

⚡ JavaScript - La magia:

JS 
// Esperar a que la página cargue
document.addEventListener('DOMContentLoaded', function() {
  // Crear las partículas
  const flux = new FluxJS({
    container: document.getElementById('fondo-particulas'),
    count: 80,
    color: '#4fc3f7',
    size: { min: 1, max: 3 },
    connections: { enabled: true, distance: 120 }
  });
});

2 Solo en una sección 🎯

¿Qué hace? Añade partículas solo a una zona estratégica de tu web, como un hero section que robe todas las miradas.

📝 HTML - Estructura básica:

HTML 
<!-- Sección con partículas -->
<section class="hero-section">
  <div id="particulas-hero"></div>
  <div class="hero-content">
    <h1>¡Bienvenido!</h1>
    <p>Esta sección tiene partículas geniales</p>
  </div>
</section>

<!-- Resto de tu página sin partículas -->
<section>
  <h2>Contenido normal</h2>
  <p>Sin partículas aquí</p>
</section>

🎨 CSS - Estilos para la sección:

CSS 
.hero-section {
  position: relative;
  height: 100vh; /* Altura completa de la pantalla */
  overflow: hidden;
}

#particulas-hero {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  z-index: 1;
}

.hero-content {
  position: relative;
  z-index: 2; /* Encima de las partículas */
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  height: 100%;
  text-align: center;
  color: white;
}

⚡ JavaScript - Solo para esa sección:

JS 
document.addEventListener('DOMContentLoaded', function() {
  const flux = new FluxJS({
    container: document.getElementById('particulas-hero'),
    count: 50,
    color: '#ffffff',
    size: { min: 0.5, max: 2 },
    speed: { min: 0.1, max: 0.3 }
  });
});

💡 Trucos de maestro desarrollador

📱

Responsive + IA

FluxJS detecta automáticamente tu dispositivo, nivel de batería y preferencias de accesibilidad. En móviles usa menos partículas, y si tu batería está baja, reduce automáticamente los FPS para conservar energía: batteryOptimization: true

👆

Multi-touch

Soporte completo para múltiples dedos en pantallas táctiles. Cada toque tiene su propia área de influencia: mouse: { touch: { multiTouch: true } }

⚡

Rendimiento

Si tu página va más lenta que un caracol, baja el número de partículas o desactiva las conexiones. FluxJS también incluye spatial partitioning y object pooling para máximo rendimiento.

👁️

Lazy Loading

Activa el viewport observer para que las partículas solo se inicien cuando el elemento sea visible: viewport: { enabled: true }

♿

Accesibilidad

FluxJS respeta automáticamente las preferencias prefers-reduced-motion del sistema operativo del usuario para una experiencia inclusiva.

🎨

Colores

Usa colores que hagan juego con tu diseño. Los colores claros brillan como diamantes en fondos oscuros.

📏

Z-index

Usa z-index: -1 para fondos y z-index: 1 para efectos encima del contenido.

🚀 Características que Te Harán Alucinar

FluxJS no es solo otra librería de partículas. Es el futuro de los efectos web con IA integrada y optimizaciones que ni sabías que existían.

🔋

Inteligencia de Batería

Detecta automáticamente el nivel de batería del dispositivo y ajusta el rendimiento para conservar energía. ¿Batería baja? FPS reducidos. ¿Cargando? ¡Potencia máxima!

performance.mobile.batteryOptimization: true
👆

Multi-Touch Pro

Soporte completo para múltiples toques simultáneos en pantallas táctiles. Cada dedo tiene su propia área de influencia con distancias y fuerzas optimizadas para móviles.

mouse.touch.multiTouch: true
👁️

Lazy Loading Inteligente

Las animaciones solo se inician cuando el elemento es visible en pantalla usando Intersection Observer. Ahorra recursos y mejora la experiencia de usuario.

viewport: { enabled: true }
♿

Accesibilidad Nativa

Respeta automáticamente las preferencias del sistema como prefers-reduced-motion para ofrecer una experiencia inclusiva a todos los usuarios.

Auto-detecta preferencias OS
🧠

Algoritmos Ultra Optimizados

Spatial partitioning para detección de colisiones, object pooling para gestión de memoria, y cache inteligente de colores. Rendimiento profesional sin configuración.

Automático & Transparente
🔧

Auto-Diagnóstico

Sistema inteligente que detecta problemas automáticamente y proporciona soluciones. Si algo falla, FluxJS te ayuda a solucionarlo con diagnósticos detallados.

flux.diagnose()

Estas características avanzadas funcionan automáticamente en segundo plano. Tú solo te preocupas por hacer que tus partículas se vean increíbles, nosotros nos encargamos del resto.

#bubbles-background {
  position: fixed;

  width: 100%;
  height: 100%;

const bubbles = new Flux({
  container: document.getElementById('bubbles-background'),
  count: 30,
  color: '#ffffff',
  size: { min: 5, max: 25 },
  speed: { min: 0.02, max: 0.1 },

🌌 Galaxia Rotante

Efecto hipnótico de partículas girando en espiral con gravedad central que te dejará en trance

HTML + JS 
// HTML
<div id="galaxy-container"></div>

// CSS
#galaxy-container {
  width: 100%;
  height: 500px;
  background: radial-gradient(circle, #1a0a2e, #000);
  border-radius: 12px;
}

// JavaScript
const galaxy = new Flux({
  container: document.getElementById('galaxy-container'),
  count: 200,
  color: {
    type: 'gradient',
    gradient: ['#8B00FF', '#4B0082', '#9400D3', 
              '#8A2BE2', '#DA70D6', '#FF69B4']
  },
  size: { min: 1, max: 4, pulse: true },
  speed: { min: 0.05, max: 0.3 },
  opacity: { min: 0.3, max: 0.8, flicker: true },
  connections: { 
    enabled: true, 
    distance: 80, 
    opacity: 0.3 
  },
  animation: { 
    type: 'orbit', 
    orbit: { radius: 120, speed: 0.012 } 
  },
  effects: { 
    glow: true, 
    glowColor: '#8B00FF', 
    twinkle: true 
  },
  mouse: { interact: true, attraction: 0.1 }
});

Manual del domador de partículas 📚

Los secretos del control absoluto

JS 
{
  container: element, // Contenedor
  count: 50, // Número de partículas
  color: '#ffffff', // Color
  size: { min: 1, max: 3 }, // Tamaño
  speed: { min: 0.1, max: 0.5 }, // Velocidad
  opacity: { min: 0.3, max: 0.8 }, // Opacidad
  connections: { // Conexiones
    enabled: true,
    distance: 100
  },
  mouse: { // Interacción
    interact: true,
    distance: 150
  }
}

Comandos para ser el jefe

JS 
// Control básico
flux.start(); // Iniciar
flux.stop(); // Detener
flux.destroy(); // Eliminar

// Configuración dinámica
flux.setColor('#ff0000');
flux.setCount(100);
flux.updateConfig({
  speed: { min: 1, max: 2 }
});
